# dev-toolbox

A collection of personal scripts and utilities I use in my daily development workflow.

## Contents

- `git/` — Git aliases and helper scripts
- `shell/` — Shell utilities and prompt configuration
- `scripts/` — Automation scripts for common tasks

## Usage

Clone the repo and source the files you need, or copy them directly into your environment.

```bash
git clone https://github.com/picorelli/dev-toolbox.git
```
